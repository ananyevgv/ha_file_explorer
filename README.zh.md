## 安装

安装完成重启HA，刷新一下页面，在集成里搜索`文件管理`即可

[![Add Integration](https://my.home-assistant.io/badges/config_flow_start.svg)](https://my.home-assistant.io/redirect/config_flow_start?domain=ha_file_explorer)

## 功能

- [x] 创建文件或文件夹
- [x] 删除文件或文件夹
- [x] 上传文件或文件夹
- [x] 重命名文件或文件夹
- [x] 修改文件内容
- [x] 支持HA主题跟随
